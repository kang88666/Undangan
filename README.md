# Undangan - Invitation Management System

Sistem manajemen undangan dengan fitur:
- Pembuatan undangan per link unik
- Management undangan terpisah
- Database Supabase
- Authentication
- Real-time updates

## Fitur Utama
1. **Event Management** - Buat dan kelola acara
2. **Invitation Links** - Setiap undangan memiliki link unik
3. **RSVP Tracking** - Pantau status konfirmasi
4. **Guest Management** - Kelola data tamu
5. **Analytics** - Laporan undangan dan kehadiran
